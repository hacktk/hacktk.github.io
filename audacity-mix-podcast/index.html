Audacityで複数のmp3ファイルをMixしてPodcastをつくる手順

2019-09-09

前提

  • 手元に複数のmp3ファイルがあり、それらをMixして1つのmp3ファイルにする作業の手順です
  • 音源は音楽ではなく人の会話です
  • 環境は macOS(Mojave), Audacity 2.3.2 です
  • 画像を入れたほうが分かりやすいのですが、長くなりすぎるので入れていません
  • あくまで自分用の備忘録なので、説明しないコンテキストがあります

手順

  1. 編集の準備をする
  2. 全体に対する編集をする
  3. 細かい編集をする
  4. Mixして出力する

編集の準備をする

音源を読み込ませ、この後の編集のために表示を見やすく変更する。

  1. 全ての音声ファイルをAudacityに読み込ませる
    • 普通にD&DでOK
    • 15分〜20分のファイルだと数秒で終わる
  2. 波形の表示を切り替えて見やすくする
    • Shift + M のメニューから、Waveform(db)にする
      • Preferences - Tracks - Default view mode でデフォルトにできる
    • trackの縦幅を目盛りが省略されずに表示されるまで広げる(マウス使用)
    • 目盛りが0.1秒刻み(数字は0.5秒刻み)の表示になるまで拡大する

全体に対する編集をする

音量、ノイズのざっくり編集を最初にやっておく。

  1. Compressorをかける
    1. 全trackを選択し、Effect - Compressor を選択
    2. 以下の設定で実行(音割れしない程度に音量を最大化する)
      • Threshold: -10db
      • Noise Floor: -40db
      • Ratio: 10:1
      • Attack Time: 0.10secs
      • Release Time: 1.0secs
      • Make-up gain for 0 db after compressing: check
      • Compress based on Peaks: check
  2. Noise Reductionをかける
    1. ノイズのみの(人が喋っていない)場所を見つけて、範囲選択する
    2. Effect - Noise Reduction を選択
    3. Get Noise Profile ボタンをクリック
      • これで「ノイズと判定するサンプル」が登録され、Profileを上書きするまで有効になる
    4. 全trackを選択し、再度 Effect - Noise Reduction を選択
    5. 以下の設定で実行
      • Noise reduction (db): 12
      • Sensitivity: 6.00
      • Frequency smoothing (bands): 3
      • Noise: Reduce
  3. ファイルに試し出力
    1. File - Export - Export as MP3
    2. 以下の設定で出力
      • Bit Rate Mode: Constant
      • Quality: 112kbps
    3. 出力されたファイルを再生し、あまりにもおかしければ調整する
  4. プロジェクトに保存する
    1. ⌘ + S で保存しておく

細かい編集をする

各trackの細かい編集をしていく。気をつけることは以下。

  • そのとき話している人以外のtrackは基本的に音声を消す(無音化)
  • 長すぎる無音時間はカットする
    • 最長でも1秒
  • カットする際、なるべく全trackを同時にカットする
    • 1つずつやると忘れが発生する
  • カットする際、声の末尾はなるべく0dbの位置で切る
    • そうでないと「ブチッ」と切れたように聞こえる
    • 声の始まりはそうでもない
  • trackの時間がズレている場合、以下の基準を使う
    • 「ある発言の最後」と「次の発言の最初」の間隔で補正する
    • 通常: 0.3秒
    • ちょっと間がある: 0.5秒
    • 食い気味: 0.1〜0.2秒
  • 相槌(主に自分がやっている)は適度にカットする
  • 「あの〜」とか「えーと」とかのフィラーは適度にカットする
  • リップノイズはなるべくカットする
  • Podcast用RSSのdurationに設定するのが1秒単位なので、端数が出ないように調整する

Mixして出力する

試し出力のときと同じ。出力後は必ず通して聴き、おかしい部分がないかチェックする。

(メモ) よく使うショートカットキー

Undo, Redo, Cut, Copy, Pasteは当然よく使うので、その他をメモ。

  • Zoom In: ⌘ + 1
  • Zoom Normal: ⌘ + 2
  • Zoom Out: ⌘ + 3
  • 選択位置から再生開始: Space
  • 選択範囲の境界を0db位置に補正: Z
  • 選択範囲を無音化: ⌘ + L

(メモ) Podcast用メタデータ編集

ツールは Kid3 を使う。
brew install kid3 でインストール。以下メモ。

  • MP3のTag2(ID3v2.3)を編集する
    • Preferences - Tags - ID3v2 - Text encoding を UTF8 にしておく
  • Title, Artist, Album, Date を入力する
  • Artworkを右側の表示欄にD&Dで設定する
    • 設定で Text Encoding を ISO-8859-1 にする(MacのFinderで表示させるため)
    • D&Dのあと、同じ表示欄をダブルクリックで設定ウィンドウが開く
